# SteamSwitcher

SteamSwitcher - это приложение для удобного переключения между аккаунтами Steam с графическим интерфейсом на WinForms.

## Описание

Приложение позволяет:
- Управлять несколькими аккаунтами Steam
- Переключаться между аккаунтами одним кликом
- Автоматически закрывать и перезапускать Steam с выбранным аккаунтом
- Настраивать темы интерфейса (светлая/темная)
- Логировать действия в реальном времени

## Архитектура

Проект состоит из следующих компонентов:

### SteamSwitcherCore
Базовая логика приложения:
- `SteamAccountManager.cs` - основной класс для управления аккаунтами и процессами Steam
- Загрузка конфигурации из `steamconfig.json`
- Управление процессами Steam (закрытие и запуск с параметрами)

### SteamSwitcherGUI
Графический интерфейс:
- `Form1.cs` - основная форма с кастомным дизайном
- Кастомные элементы управления (список аккаунтов, кнопки с иконками)
- Поддержка тем (светлая/темная)
- Лог-панель для отображения действий

### SteamSwitcherTests
Автотесты:
- Unit-тесты для `SteamAccountManager`
- Интеграционные тесты для конфигурации и взаимодействия компонентов
- Тесты на корректность загрузки конфигурации и переключения аккаунтов

## Установка и настройка

1. **Клонируйте репозиторий:**
   ```bash
   git clone <repository-url>
   cd SteamSwitcher
   ```

2. **Восстановите зависимости:**
   ```bash
   dotnet restore
   ```

3. **Настройте конфигурацию:**
   - Отредактируйте файл `steamconfig.json` в корневой папке проекта
   - Укажите путь к `Steam.exe`
   - Добавьте ваши аккаунты в формате:
     ```json
     {
       "SteamPath": "C:\\Program Files (x86)\\Steam\\steam.exe",
       "Accounts": {
         "Основной аккаунт": "username1",
         "Альтернативный": "username2"
       }
     }
     ```

## Запуск

### Быстрый запуск (только сборка и запуск)
```bash
# Из корневой папки проекта
.\Run.bat
```

Этот скрипт:
1. Собирает проект в Release конфигурации
2. Запускает GUI-приложение

### Полный цикл: тесты + сборка + запуск
```bash
# Из корневой папки проекта
.\Deploy.ps1
```

Этот скрипт:
1. Запускает все автотесты
2. Если тесты пройдут - собирает проект в Release конфигурации
3. Запускает GUI-приложение

### Параметры скрипта Deploy.ps1
- `-SkipTests` - пропустить запуск тестов
- `-SkipBuild` - пропустить сборку проекта
- `-SkipLaunch` - пропустить запуск приложения

Примеры:
```bash
# Только тесты
.\Deploy.ps1 -SkipBuild -SkipLaunch

# Только сборка и запуск (без тестов)
.\Deploy.ps1 -SkipTests

# Только запуск (без тестов и сборки)
.\Deploy.ps1 -SkipTests -SkipBuild
```

### Ручной запуск
```bash
# Сборка
dotnet build --configuration Release

# Запуск GUI
.\SteamSwitcher\SteamSwitcherGUI\bin\Release\net10.0-windows\SteamSwitcherGUI.exe
```

### Запуск тестов отдельно
```bash
dotnet test
```

## Использование

1. Запустите приложение через `Deploy.ps1`
2. Выберите аккаунт из списка
3. Нажмите кнопку "Switch Account"
4. Приложение автоматически закроет текущий Steam и запустит новый с выбранным аккаунтом
5. Используйте кнопку темы для переключения между светлой/темной темой
6. Следите за логами в нижней панели
